A simple yet engaging clicker game built with vanilla JavaScript, HTML, and CSS.

## Features

- **Click to Earn**: Click the main button to earn points
- **Passive Income**: Purchase upgrades to earn money per second
- **Progressive Upgrades**: Unlock new upgrades as you progress
- **Efficiency Tracking**: See the efficiency ratio for each upgrade to make informed purchases
- **Responsive Design**: Works on desktop and mobile devices

## How to Play

1. Click the "Click Me!" button to earn points
2. Use your earnings to purchase upgrades:
   - **Click Power**: Increases points gained per click
   - **Auto Clicker**: Generates passive income automatically
   - **Worker**: Increases passive income further
   - **Factory**: High-cost, high-reward passive income upgrade
3. New upgrades become visible once you can afford them
4. Once you purchase an upgrade, it remains visible for future purchases

## Upgrades

Each upgrade shows:
- **Name**: What the upgrade does
- **Production**: How much it produces per click or per second
- **Owned**: How many you currently own
- **Efficiency** (Passive upgrades only): Production per cost ratio to help you decide which upgrades offer the best value
- **Cost**: How much it costs to purchase
